# 🎓 EduConnect Hub

EduConnect Hub is a user-friendly communication platform designed for academic departments. It serves as a centralized space that bridges the gap between students and faculty, allowing users to get answers to critical questions, find collaboration partners, and manage project mentorships efficiently.

---

## 🚀 Key Challenges Solved

* **Finding Help:** Provides an avenue for students to get rapid, accurate answers to academic and administrative queries.
* **Project Collaboration:** Simplifies the process of finding peer teammates and faculty mentors for research or development projects.
* **Centralized Communication:** Eliminates fragmented email threads and scattered announcement boards by bringing department communication into one single hub.

---

## ✨ Features

* **Smart Q&A Matching:** Post academic or technical questions and get matched with relevant department experts or faculty members.
* **Project Collaboration Space:** Share project concepts, outline required skills, and dynamically find matching teammates.
* **Real-time Chat:** Instant messaging capabilities for seamless, immediate collaboration on tasks and inquiries.
* **Role-Based Secure Access:** Protected system utilizing secure authentication, ensuring customized interfaces and access privileges for both students and faculty members.
* **Modern, Responsive Interface:** Clean, highly-intuitive design optimized for both desktop and mobile devices.

---

## 🛠️ Technologies Used

### Frontend
* **React** (v18+)
* **TypeScript** / **JavaScript**
* **Tailwind CSS** (for responsive utility-first styling)

### Backend & Database
* **Node.js** & **Express.js**
* **PostgreSQL** (Relational Database)
* **Prisma ORM** (Database management and type-safe queries)

### Security & Communication
* **JSON Web Tokens (JWT)** (Secure authentication and state-less session management)
* **WebSockets / Socket.io** (For real-time chat capabilities)
